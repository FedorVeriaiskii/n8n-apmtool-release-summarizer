# APM-Tool Release Summarizer Agent

## Overview

The APM-Tool Release Summarizer Agent is an intelligent n8n workflow designed to automatically fetch, analyze, and summarize APM-Tool APM tool release notes. This automation solution provides stakeholders with concise, well-structured summaries of the latest releases across multiple APM-Tool components.

### Key Features

- **Multi-Component Support**: Automatically processes release notes for various APM-Tool components including OneAgent, ActiveGate, APM-Tool API, and APM-Tool Operator
- **AI-Powered Summarization**: Leverages OpenAI GPT models to extract key information and generate structured summaries categorized by breaking changes, announcements, new features, and resolved issues
- **Automated Email Reports**: Generates professional HTML email reports with formatted release summaries
- **Form-Based Triggering**: Easy-to-use form interface for selecting specific APM-Tool components to analyze
- **Intelligent URL Discovery**: Automatically finds the latest release documentation URLs for each component

### Use Cases

- **DevOps Teams**: Stay informed about critical updates and breaking changes in APM-Tool tools
- **Release Management**: Track feature releases and bug fixes across APM components  
- **Technical Documentation**: Maintain organized records of release information
- **Stakeholder Communication**: Provide non-technical teams with digestible release summaries

## Workflow Description

The APM-Tool Release Summarizer Agent follows a sophisticated multi-stage process to automatically gather, analyze, and distribute APM-Tool release information.

### Workflow Stages

#### 1. **Form Trigger & Component Selection**
- **Release Notes Form Trigger**: Initiates the workflow through a user-friendly form interface
- **Split APM-Tool Elements for Processing**: Processes user-selected APM-Tool components for analysis
- **Loop Through Selected Components**: Iterates through each selected component individually

#### 2. **URL Discovery & Content Retrieval**
- **AI Agent ‚Äî Find Latest Release URL**: Uses OpenAI GPT to intelligently discover the most recent release documentation URLs
- **Structured Output Parser (URL Finder JSON)**: Ensures proper formatting of discovered URLs
- **Component-Specific HTTP Requests**: Fetches content from dedicated pages:
  - üåê **HTTP Request ‚Äî APM-Tool Managed Page**: Main APM-Tool documentation hub
  - üåê **HTTP Request ‚Äî OneAgent Page**: OneAgent-specific release notes
  - üåê **HTTP Request ‚Äî ActiveGate Page**: ActiveGate component updates
  - üåê **HTTP Request ‚Äî APM-Tool API Page**: API changes and improvements
  - üåê **HTTP Request ‚Äî APM-Tool Operator Page**: Kubernetes operator releases

#### 3. **Content Processing & Routing**
- **Route by Selected APM-Tool Component**: Intelligent routing based on component type
- **Extract HTML Body and Component**: Extracts relevant content from fetched web pages
- **HTTP Request (AI Tool Access)**: Provides AI agents with web scraping capabilities

#### 4. **AI-Powered Analysis & Summarization**
- **AI Agent ‚Äî Summarize Release Notes**: Advanced AI analysis of release documentation using OpenAI GPT models
- **Structured Output Parser (Summary JSON)**: Formats summaries into structured JSON with categories:
  - **Breaking Changes**: Critical updates requiring immediate attention
  - **Announcements**: Important notifications and deprecations  
  - **New Features**: Feature additions and improvements
  - **Technology Support**: Platform and integration updates
  - **Resolved Issues**: Bug fixes and performance improvements

#### 5. **Report Generation & Distribution**
- **Aggregate Summarized Release Data**: Combines individual component summaries
- **Generate HTML Summary Report**: Creates professional HTML formatting
- **Create Final HTML Document**: Assembles the complete report with styling
- **Send Release Summary Email**: Delivers formatted reports via Gmail integration
